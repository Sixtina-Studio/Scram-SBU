# Scram-SBU
Scram SBU is an strategy game set in the Middle East, the United States of America has appointed you as the Captain of the USS Abraham Lincoln plane-carrier, lead the aerial offensive across 10 missions, scram your jets, improve them and fight the Iranians at operation Epic Fury.

SCRAM SBU is a Shell Based UI game, it has no fancy graphics, its just commands on a terminal, this project was born after the need of learning OOP.
